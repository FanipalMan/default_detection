## Installation

1. Create and activate a Python environment (recommended: Python 3.11+).
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Project structure

- data/ - input datasets used for training and evaluation
- utilities/ - helper modules for constants, preprocessing, and metrics
- eda.ipynb - exploratory data analysis notebook
- feature_selection.ipynb - feature selection experiments
- training_inference.ipynb - models training and inference workflow
- requirements.txt - Python dependencies
